## Decisions & surprises
- **Cold start exceeds the tool timeout.** The first call (model load onto GPU) timed out on the
  MCP response window but completed in the background; the warm re-fire returned immediately.
  `keep_alive -1` keeps it loaded thereafter.
- **8B sign-interpretation failure → prompt rule.** The first prompt passed a *signed* grid value
  ("negative = exporting"); the model wrongly said "exporting" while importing. Pre-computing the
  direction in HA fixed it. **Rule adopted:** compute facts in HA, let the model only phrase them
  (ADR-0001). Documented in the prompt file.
- Direct service/API calls do **not** render Jinja — pre-render for ad-hoc tests; automations render normally.
